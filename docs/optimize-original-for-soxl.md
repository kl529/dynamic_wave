## 결론

1. **원본 양수 buyTarget은 SOXL에 완전 부적합** — 학습 TOP 30 내 0개
2. **최적 buyTarget: -3% ~ -5%** — 현재 v3 방향이 맞았음
3. **주요 개선 포인트**: safe 매도목표 0.8%→2.0%, 보유기간 30→20일, 공세 매도 6%→8%, 공세 보유 10→7일
4. **10분할 유지**: 7분할보다 일관적으로 유리
5. **Overfitting 경고**: 학습/검증 간 최적 조합이 일치하지 않음 → 보수적 중간값 선택 필요
