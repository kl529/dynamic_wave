## 1. 동파법의 철학

### 핵심 원리: 역추세(Mean Reversion) + 분할 매수

동파법은 한 가지 믿음에서 출발한다.

> **"가격은 크게 떨어지면 반드시 올라온다. 그 반등의 일부를 안전하게 챙긴다."**

이를 구현하는 구조:

```
1. 분할: 자본을 N등분해 리스크 분산
2. 매수 조건: 전일 대비 -X% 이상 하락한 날에만 진입
3. 매도 조건: 평균 매수가 대비 +Y% 반등 시 즉시 청산
4. 손절: 최대 보유기간(N거래일) 초과 시 시장가 청산
5. 모드 전환: RSI 기반으로 시장 상황에 맞게 공격성 조절
```

수익 공식은 단순하다:

```
기대값 = (승률 × 평균수익%) - (패율 × 평균손실%)

수익이 나려면: 승률이 충분히 높고, 손절 손실이 작아야 한다
```

전략이 잘 작동하는 시장:
- **박스권**: 하락 → 매수 → 반등 → 매도의 사이클이 반복됨
- **변동성이 높되 방향이 없는 장**: 하락일과 반등일이 번갈아 나옴

전략이 안 되는 시장:
- **강한 상승 추세**: 하락일 자체가 없어서 진입을 못 함 (기회 손실)
- **강한 하락 추세**: 진입 후 계속 하락, 반등 없이 손절만 반복됨

## 4. 구조적 문제 진단

### 왜 상승장에서 못 버는가

```
SOXL이 강하게 상승하는 구간 (예: 2023 +235%):

1. 전일 대비 -3% 하락일이 거의 없음
   → 안전모드: 진입 기회 자체가 없음

2. 진입하더라도 +0.8% 도달 즉시 청산
   → 이후 수백% 추가 상승에 무관

3. 결과: 자본 대부분이 현금으로 유휴
   → 시장 +235%인데 전략 -7.6%
```

### 왜 횡보장(2012)에서도 손실인가

```
2012년 SOXL: 연간 B&H 0% (시작 $0.45, 마감 $0.45)

하지만:
  - 연중 등락폭은 상당했음 (최고 $0.70, 최저 $0.25)
  - 전략이 고점 구간에서 진입 → 저점까지 하락 → 30일 만료 손절 반복
  - 반등 기회에 이미 현금 부족 or 순번 대기 중

→ B&H는 ±0이지만 전략은 -18.6%
```

이는 역추세 전략의 고유한 취약점이다:
- **하락 추세 중 반등을 기대하고 진입했지만, 추세가 더 이어질 때** 손절이 누적된다.
- 이를 해결하려면 "추세 방향성 필터"가 필요하다.

### 근본 원인 정리

```
동파법의 3가지 근본 취약점:

1. 추세 장(상승/하락 모두)에서 역추세 베팅
   → 상승 추세: 기회 손실
   → 하락 추세: 반복 손절

2. 고정 매도 목표(+0.8%)
   → 상승 추세에서 너무 빨리 청산
   → 이후 수익 참여 불가

3. 분산이 리스크 축소에만 초점
   → 수익 증폭 메커니즘이 없음
   → 작은 수익 × 많은 횟수 구조는 손절 1회로 쉽게 상쇄됨
```

## 5. 개선 방향 논의

### 전제: 동파법의 포지셔닝 재정의

지금까지의 목표: "단독으로 연 10% 수익 달성"
**현실적 포지셔닝: "폭락장 대비 헤지 + 박스권 알파"**

2010~2026 데이터가 보여주는 것:
- 단독 전략으로는 B&H를 이길 수 없음 (16년 중 3년만 우위)
- 하지만 폭락장 방어력은 탁월 (2022: B&H -87% vs 동파법 -36%)
- 포트폴리오의 일부로 활용 시 의미 있음

---

### 방향 A: 즉시 개선 (분할 수 증가)

**변경**: div=5 → div=10
**효과**: 모든 구간에서 손실 20~25%p 감소 확인됨
**부작용**: 거래당 금액 감소로 절대 수익액 감소

---

### 방향 B: 상승 추세 참여 (bull 모드 강화)

현재 bull 모드: RSI 55~65, 상승 중 → 매수 -1%, 매도 +10%, 보유 45일
**문제**: -1% 하락도 없는 강한 상승장에서는 bull 모드도 진입 못 함

개선안:
```
bull 모드 (RSI 55~65 상승 중):
  기존:  전일 대비 ≤ -1% 일 때 매수
  개선:  전일 대비 ≤ +1% 이면 매수 (약간의 상승일에도 진입 허용)
         → 상승 추세 중 눌림목 없어도 분할 진입
```
**리스크**: 고점 추격매수 위험 증가, 수익이 기존보다 나빠질 수 있음

---

### 방향 C: 하이브리드 (B&H 레이어 추가)

전략 구조를 2레이어로 분리:
```
레이어 1 (70%): 항상 SOXL 보유 (B&H)
레이어 2 (30%): 동파법 분할 매수/매도

역할 분리:
  레이어 1: 상승장 수익 참여 (SOXL 상승 시 이익)
  레이어 2: 폭락장 추가 매수 / 변동성 알파 생성
```
**리스크**: 폭락장에서 레이어 1(70%)이 그대로 손실

---

### 방향 D: 시장 체제 분류 후 전략 전환 (권장)

```
RSI 주간 기준 + 추가 필터로 4가지 체제 구분:

체제 1 — 강한 상승 (RSI > 70, 상승 추세):
  → SOXL 전체 보유 + 동파법 비활성화

체제 2 — 완만 상승 (RSI 50~70, 상승 추세):
  → bull 모드 (매수 조건 완화: ≤ +0%)

체제 3 — 횡보/불확실 (RSI 40~60, 방향 없음):
  → 현재 safe/aggr 혼합 모드 유지

체제 4 — 하락 (RSI < 40, 하락 추세):
  → 매수 중단, 기존 포지션 손절 후 현금 대기 (cash 모드)
```

**장점**: 각 시장 상황에 최적화된 대응 가능
**단점**: 체제 전환 판단 오류 시 모드 전환 지점에서 손실 집중
