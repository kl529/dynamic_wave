## 1. 핵심 아이디어

### 문제 인식

동파법 v3.1의 구조적 한계:
- **상승장 완전 불참**: 2013 +125% B&H → 전략 +6%, 2019 +221% B&H → 전략 -6%
- 원인: 매수 임계값 -3%/-5%로 강세장에선 매수 기회가 거의 없음 (8~13% 거래일만 매수)
- 누적 수익 (2010~현재): 동파 단독 **-87%** vs B&H **+5,757%**

### B&H 아이디어 차용

**B&H = "올라갈 때 팔지 않는다"**

| B&H의 강점 | 동파법의 강점 |
|-----------|-------------|
| 추세 전부 탑승 (2013~2024 SOXL +10배+) | 하락장 방어 (+35~67%p 손실 감소) |
| 구현 단순 (그냥 보유) | RSI 기반 시장 국면 판단 |

**핵심 결합 아이디어**: RSI가 이미 시장 국면을 잘 분류하고 있다
→ RSI 상승(aggressive/bull) 구간 = B&H / RSI 하락(safe/cash) 구간 = 동파법

## 2. 하이브리드 전략 구조

```
RSI 모드 판단 (14주봉 Wilder's RSI)
    │
    ├─ aggressive / bull  →  B&H 모드 (전액 매수 & 보유)
    │
    └─ safe / cash        →  동파법 모드 (분할 매수/방어)

전환 로직:
  Dongpa → B&H: 분할 포지션 전부 청산 → 전액 재매수
  B&H → Dongpa: 전량 매도 → 분할 자금(10등분) 재배분
  → Cash:       전량 청산 → 현금 보유
```

## 3. 필터 실험 결과

### 3-1. 기본 하이브리드 문제점

기본 하이브리드는 연간 전환 횟수 14~31회로 지나치게 많음.
2022년처럼 RSI가 반등 신호를 주다가 재하락하는 패턴에서 **-59%** 손실 발생.

### 3-2. 개선 필터 4종 비교 (2010~현재)

| 전략 | 누적 수익 | 하락장 | 상승장 | 연도별 승리 |
|------|---------|--------|--------|-----------|
| 동파 v3.1 단독 | -87% | 약 | 약 | 0년/17년 |
| 기본 하이브리드 | +2.95억% | -49.7% | +8,090% | 5년/17년 |
| +최소보유 10일 (양방향) | **+1.52억%** | -47% | **+21,689%** | **9년/17년** |
| +SMA50 (진입만) | +2.18억% | **+44%** | +6,307% | 1년/17년 |
| **★최종 (비대칭)** | +1.49억% | **+59%** | +3,336% | 2년/17년 |

### 3-3. ★최종 비대칭 필터 설계

```
B&H 진입 (dongpa → bnh):
  조건 1: 가격 > SMA50 (50일 이동평균 위)  ← 상승 추세 확인
  조건 2: 마지막 전환 후 10 거래일 경과     ← 잦은 전환 방지

B&H 탈출 (bnh → dongpa/cash):
  RSI safe/cash 신호 즉시 반영              ← 빠른 위험 회피 (락 없음)
```

**비대칭 설계 이유**: 진입은 신중하게(false signal 차단), 탈출은 빠르게(손실 최소화)
→ 2022년 결과: 기본 하이브리드 **-59%** → ★최종 **+9.7%**

B&H 탈출: RSI(safe/cash) → 즉시 전환

## 7. 결론 및 한계

### 결론

1. **하이브리드 전략이 두 전략 모두를 압도**: 동파 단독(-87%)과 B&H(+5,757%) 대비 모든 하이브리드 변종이 훨씬 우수
2. **RSI가 시장 국면 분류기 역할을 충분히 수행**: aggressive/bull = 상승장, safe/cash = 방어 구간
3. **비대칭 필터가 핵심**: 진입은 신중(SMA50+대기), 탈출은 즉시 → 하락장 최고 성과 달성
4. **최종 권장**: 수익 극대화는 +최소보유, 리스크 균형은 ★최종(비대칭)

### 백테스팅 한계

- 연간 독립 시뮬레이션 (연 경계에서 포지션 초기화됨)
- 슬리피지(대량 주문 시 호가 영향) 미반영
- 전환 시 전액 매도/매수 — 실제로는 수 거래일에 걸쳐 실행 필요
- SOXL 상장(2010-03-11) 이후 데이터만 사용 — 충분한 샘플이지만 미래 보장 없음
- 과거 RSI 패턴이 미래에도 동일하게 작동할지 불확실
