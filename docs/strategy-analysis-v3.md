## 핵심 문제: 강한 상승장에서의 매수 기회 부재

### 메커니즘

```
강한 상승 추세 (SOXL 매일 +3~8%):
  → 전일 대비 -3% 하락일이 수 주~수 개월간 없음
  → 매수 신호 발생 안 함
  → 현금 5분할 전부 유휴 상태
  → 상승분 전혀 참여 못함
  → 결과: 시장 +235%인데 전략 -7.6%
```

### 2023년 구체 사례

```
2023년 SOXL: 1월 $8 → 12월 $27 (+237%)
- 상승 구간: 연속 상승 주가 많음, -3% 하락일 극히 적음
- 동파법: 매수 58회 (1년 250거래일 중 23%)
- 매수한 날 대부분: 잠깐 눌림 → 곧바로 더 오름
- +0.2% 도달 즉시 청산 → 이후 수백% 추가 상승 미참여
```

### v2와 비교해도 근본 문제는 동일

v2는 매일 매수했지만 승률이 낮아서 손실.
v3는 선택적으로 매수하지만 상승장에서 아예 못 사서 기회 손실.

## 박스권에서 괜찮은 이유

```
박스권/완만한 하락 (SOXL 매일 ±3~5% 등락):
  → -3% 하락일이 주기적으로 발생
  → 매수 기회 존재 → 평단가 확보
  → 다음 반등일에 +0.5% 목표 달성 가능
  → 작은 수익 반복 축적
  → 결과: 수수료를 감당할 수 있는 수준의 손익
```

실제로 공세 모드(안전 -5% 진입 / +2.5% 목표)는 2024(SOXL -2.6%) 기간에도 +1.88% 달성.
박스권에서 분할 매매의 원래 의도가 가장 잘 작동하는 시장.

## 전략의 구조적 한계

현재 전략은 **역추세(Mean-Reversion)** 구조:

- 하락 → 매수 (가격이 돌아올 것을 기대)
- 반등 → 매도 (작은 수익 실현)

이 구조는 **추세가 없는 시장**(박스권, 변동성 장)에서 유리하고,
**강한 추세 시장**(일방적 상승/하락)에서 불리하다.

| 구조 | 강점 | 약점 |
|------|------|------|
| 역추세 (현재) | 박스권, 변동성 | 추세장 (상승/하락 모두) |
| 추세 추종 | 상승장 | 박스권 (잦은 손절) |

## 개선 방향 후보

### 방향 A: 상승 추세 감지 → 매수 조건 완화

RSI가 강한 상승 신호일 때(공세 모드) 매수 조건을 완화:
```
공세 모드(RSI 과매도):  -5% 이상 하락 시 매수 (현재)
공세 모드(RSI 상승중):  -2% 이상 하락 시 매수 (완화)
```
- 장점: 상승장 참여율 향상
- 단점: 박스권에서 과매수 위험 증가

### 방향 B: 매도 목표 동적 조정

추세 강도에 따라 매도 목표를 키움:
```
RSI 상승 추세: 목표 +5~10% (장기 홀딩)
RSI 중립/하락: 목표 +0.5~2.5% (빠른 회전)
```
- 장점: 상승장에서 수익 확대
- 단점: 목표 미달 시 손절 기회 증가

### 방향 C: 추세 판단 필터 추가

단순 RSI 대신 이동평균선 기울기 또는 ADX로 추세 강도 판단:
```
추세 없음(ADX < 25): 현재 역추세 전략 유지
추세 있음(ADX ≥ 25): 매수 조건 없이 포지션 유지 or 신규 진입 허용
```

### 방향 D: 전략 조합

동파법을 베이스로 하되, 강한 상승장에서는 B&H 비중을 일부 유지:
```
전체 자본의 70%: 동파법 (분할 매매)
전체 자본의 30%: 무조건 보유 (B&H 성격)
```
- 장점: 상승장 소외 현상 완화
- 단점: 폭락장에서 30% 손실 불가피

## 결론

현재 v3 전략은 **"폭락 방어 특화 역추세 전략"**으로 포지셔닝됨.

- 폭락장: B&H 대비 38%p 방어 ← 명확한 강점
- 박스권: 소폭 손실 수준으로 관리 ← 허용 가능
- 상승장: 완전 소외 ← 구조적 한계, 해결 필요

**다음 단계**: 강한 상승 추세 구간을 감지해서 매수 조건을 조정하는 로직 추가 여부 검토.
RSI 기반 모드 전환이 이미 있으므로, 공세 모드에서의 매수 임계값 조정이 가장 자연스러운 개선 경로.
