## 핵심 진단

### 개선된 점

```
v2: "오늘 +3% 미만이면 매수" → 거의 매일 매수 → 하락 추세에서 지속 물타기
v3: "오늘 -3% 이상 하락해야 매수" → 큰 하락일에만 진입
→ 매수 횟수 46회 (v2: 161회) — 3.5배 감소
→ 수수료 $86 (v2: $222) — 61% 절감
→ MDD -51.5% (v2: -65.1%) — 13.6%p 개선
```

### 여전한 문제: 강한 상승장 대응 불가

```
SOXL이 강하게 상승하는 구간:
1. 매수 조건 "change ≤ -3%" → 하락일이 거의 없어 포지션 진입 자체가 적음
2. 진입하더라도 +0.2% 달성 즉시 청산 → 추가 상승 참여 불가
3. 결과: 2023년(+235%) 같은 강세장에서 -7.6% 손실
```

### 핵심 강점: 폭락장 방어 극적 강화

```
2022년 SOXL -86.6% 구간:
- v2 AUTO: -78.3% (B&H 대비 +8.3%p)
- v3 AUTO: -48.3% (B&H 대비 +38.3%p) ← 30%p 추가 방어
```

## 결론

| 시장 환경 | v2 | v3 |
|-----------|----|----|
| 강한 상승장 | ❌ 매우 불리 | ❌ 불리 (개선됨) |
| 횡보장 | ❌ 불리 | △ 소폭 개선 |
| 하락장 | ✅ B&H 대비 우위 | ✅✅ B&H 대비 **대폭** 우위 |

**v3 전략 특성**: 하락일에만 진입하는 구조 덕분에 폭락장 방어력이 비약적으로 향상. 그러나 추세 상승장에서는 매수 기회 자체가 줄어 소외될 위험 존재.

**다음 개선 방향**: 상승 추세(공세모드)에서의 매수 조건 완화 또는 매도 목표 확대를 통해 추세장 참여율 향상 검토.
